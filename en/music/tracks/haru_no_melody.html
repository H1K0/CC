Tune 春のメロディー (Haru no Melody)

Info

Composer, arranger, programmer, cover designer: H1K0

Key: C-dur

Tempo: 110 BPM

Release date: 2017-03-08

Description

The spring has come!

Actually spring is one of my favorite seasons (exclusive of the school exams)! It's so cute and romantic! The first flowers bloom, the snow runs down... I'm so inspired by that! Hope this spring I'll make more tracks than previously!

There're three parts in this project. Each of them describes spring. For example, eavesdrop, sakura blossom, weather and so on. But you can imagine other things!

Enjoy!)
